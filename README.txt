WAY OF THE BELL - FLAT GITHUB PAGES PACKAGE

All files in this folder belong in the root of your GitHub repository.
There is no assets folder.

UPLOAD
1. Open the Tets repository on GitHub.
2. Choose Add file > Upload files.
3. Upload every file from this folder.
4. Allow index.html, manifest.json and service-worker.js to replace the old versions.
5. Commit the changes.

FILES THAT MUST BE PRESENT
- index.html
- manifest.json
- service-worker.js
- brand-logo.png
- scene-temple.jpg
- scene-carry.jpg
- favicon-64.png
- icon-192.png
- icon-512.png
- icon-maskable-512.png

After GitHub Pages republishes, these URLs should work:
